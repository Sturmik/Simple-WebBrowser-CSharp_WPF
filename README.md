# Simple-WebBrowser-CSharp_WPF
Simple web browser written on WPF

This app, allows you to surf the internet with basic functions, like:
- Close tab
- Redo
- Undo
- Home (google.com)
- Tag as favourite
- Refresh
- Enter adress of website
- Check favourite sites
